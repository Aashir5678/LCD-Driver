# LCD-Driver
Arduino code to control an LED

Made using data sheet for HD44780 controller: https://www.crystalfontz.com/controllers/datasheet-viewer.php?id=97

## Wiring for LCD

<img width="1164" height="696" alt="image" src="https://github.com/user-attachments/assets/e8d0690e-e29a-4aa1-aed4-ccf53f0e632e" />


VDD: GND

VSS: +5 V

Vo: Output from 10k potentiometer

RS: Pin 13 on Arduino

RW: Pin 12 on Arduino

ENABLE: Pin 11 on Arduino

D4 - D7: Pins 6 - 3 on Arduino

A: +5 V

K: GND
